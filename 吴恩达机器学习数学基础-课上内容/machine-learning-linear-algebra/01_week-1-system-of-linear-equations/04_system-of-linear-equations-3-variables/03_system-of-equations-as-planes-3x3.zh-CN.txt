当你学习二乘二矩阵时，你会看到两个变量的线性方程 可以用平面中的线来表示。 以下是表示方程 a + b = 10 和 a + 2 b = 12 的直线。 同样，三个变量中的线性方程由 三维空间中的平面表示。 在右边，你有带有三轴的三维空间。 轴 a 是水平轴，轴 b 是垂直轴 ，轴 c 应该从屏幕一直延伸到你的鼻子。 那么，例如，方程 a + b + c = 1 的图在太空中会是什么样子。 好吧，让我们来看看一些属于这个情节的要点。 例如，点 1,0,0 属于此图，因为 1 + 0 + 0 等于 1。 这就是 a 协调员 1 和另一个指向你的时候。 0,1,0 也属于这里，因为 0 + 1 + 0 = 1。 最后 0,0,1 也属于这个地块。 现在，三个点定义了一个平面，实际上 穿过这三个点的整个平面就是方程 a + b+ c = 1 的解集。 因此，就像具有两个变量的线性方程对应于 平面中的一条直线一样，由3个变量组成的线性方程对应于空间中的平面。 现在，在常数方程为零的特殊情况下 ，例如在方程 3a-5b + 2c = 0 中， 平面必须穿过原点，即 0,0,0。 其原因是，如果我们设置 a = 0 b = 0 和 c = 0，则这是方程的解，因为 3 (0) + 5 (0) + 2 (0) 的总和 = 0。 因此，现在就像我们过去用相交线获得点作为 系统方程解的方法一样，你也可以相交平面，看看这个。 因此，我们不太关心获得正确的可视化效果，因为 这些东西很难在二维空间中可视化。 但是，我们将担心这些 飞机的交叉点是如何出现的。 因此，对于这个方程组，a + b+ c = 0，a + 2b + c = 0，a + b + 2c = 0。 让我们来看看第一个方程， 它对应于一个穿过 0,0,0 的平面。 第二个方程对应于其他穿过 0,0,0 的平面，这两个平面在一条直线上相交。 第三个对应于另一个经过点0,0,0的平面， 三个平面在一个点相交，恰好是点0,0,0。 所以这很重要，因为这是一个非奇异系统，所以它有一个独特的解决方案。 而这个独特的解决方案是 0,0,0 点。 现在让我们看一下系统 2，第一个方程又是一个平面， 第二个方程是另一个平面，它们都穿过原点并 相交了一条线。 现在这个系统是单数的，所以 发生的事情是第三个平面也穿过了另外两个平面和 原点，但它实际上穿过了另外两个平面，再加一条线。 因此，这是三架飞机，它们都经过同一条线。 因此，这组解不仅仅是一个点，它是一条整线，所以 方程组有多个解， 这意味着该系统是单数的。 最后我们有了另一个系统，其中的方程是 a + b + c = 0、 2a + 2b + 2c = 0 和 3a + 3b + 3c = 0 和 3a + 3b + 3c = 0。 第一个对应于一架飞机。 现在，正如你之前所看到的，第二个方程只是第一个方程的倍 数，所以它实际上是同一个方程。 因此，它对应于完全相同的飞机也就不足为奇了。 第三个方程也对应于完全相同的平面。 因此，系统的解集合是平面中的每个点， 有多个解，因此该系统又是单数的
