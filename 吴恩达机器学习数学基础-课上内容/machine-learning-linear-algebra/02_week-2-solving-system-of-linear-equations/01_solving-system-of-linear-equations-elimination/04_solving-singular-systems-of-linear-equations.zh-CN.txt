现在这个方法实际上比你想象的要强大。 在前面的示例中，您看到了它对 具有独特解决方案的非单一系统的作用。 但是，对于单一系统，即没有 独特解决方案的系统，会发生什么。 让我们求解方程组 a + b = 10 和 2a + 2b = 20 之前也出现的奇异系统。 回想一下，该系统是多余的，因为第二个方程等同 于第一个方程。 因此，让我们尝试 和以前相同的步骤，尝试从第二个方程中删除 a，看看它会带我们走向何方。 第一步是将两个方程除以 a 的系数，得到两个方程 a + b = 10 和 a + b =10。 请注意，它们是一样的。 下一步是从方程 a + b = 10 中减去 方程 a + b = 10，从第二个方程中删除 a。 现在你在这里得到了什么？ 好吧，你得到 0 = 0，因为你要从自身中减去整个字符串。 这个方程式 0 = 0 是微不足道的， 不幸的是，你没有得到关于 b 可能是什么的信息。 发生了什么？ 好吧，在你努力从第二个方程中删除 a 或者 不理会 b 的过程中，你也移除 b。而且你无能为力获得 b。你无法进行任何操作来只删除其中一个变量， 因为你为删除一个变量所做的任何事情也会删除另一个变量， 因为系统是单数的。 再说一遍，这是因为第二个方程没有为系统增加任何价值。 因此，换句话说，求解后的系统是 a + b = 10 并且 没有其他方程可以为我们提供更多信息。 那么你还能得到看起来像 a 等于某物而 b 等于其他东西的解吗？ 好吧，你仍然可以选择任何数字 x 然后让 a 等于 x。 那么 b 是多少？ 好吧，如果 a + b = 10，那么 b 必须 是 10-x。这是系统的解决方案。 这不是一个唯一的解决方案，因为 x 可以是任何值。 因此，求解系统有一个自由度，如果你改变这个 自由度 x，你可以得到许多不同的系统解。 例如，解决方案如你之前所说的那样形成一条线。 现在，如果你对 矛盾的单数系统使用相同的方法会发生什么？ 回想一下，你 之前看到的矛盾系统是这个 a + b = 10 和 2a + 2b = 24。 让我们用同样的步骤来解决这个问题。 首先，我们要从这个方程中删除 a。 因此，我们将两个方程除以 a 的系数。 我们从方程 2 中减去方程 1 ，得到左边的 0 n 等于右边的 2。 0 永远不等于 2 所以这是矛盾的。 因此，该系统没有解决方案。 所以，现在你已经准备好参加测验了。 测验如下。 你能否求解以下方程组 5a + b = 11 和 10a + 2b = 22。 因此，如果你仔细观察系统中的两个方程， 你会发现方程2是方程1乘以二。 或者等效，如果你取方程2并将其除以2，则得到方程1。 因此，该系统是单数的，它有无限多的解。
