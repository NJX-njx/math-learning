现在你已经学会了如何用两个变量求解两个方程组， 让我来向你展示如何用三个变量求解三个方程组。 其实非常相似。 这里的目标首先是单独离开 a。 因此，我们将确保唯一包含 a 的方程 是第一个方程，而第二个和第三个方程上没有 a。 因此，方法是对第一列进行归一化， 即将每行除以 a 的系数，以确保 每个方程的变量 a 的系数均为 1。 现在使用第一个方程将变量 a 从所有其他方程中移除。 因此，从第二个方程中减去第一个方程， 然后从第三个方程中减去得到以下结果。 现在，在生成的系统中，你已经成功地分离出了 a， 剩下的是一个由两个方程组成的系统，其中有两个未知数，分别是 b 和 c。而且 你确切地知道如何求解这两个方程。 所以当你求解它时，你会得到 b 和 c 的值。 让我来告诉你怎么做，所以现在让我们忘记第一个方程， 求解第二个和第三个方程，你就知道该怎么做了。 首先，将这两行除以 b 的系数 ，得出 b 的前导系数现在为 1。 现在使用这个方程，第二个方程从第三个方程中删除 b。 因此，从第三个方程中减去第二个方程即可得出， 11 减去 11/6，c 等于减去 11/2。 这样，你就在底部的两行中隔离了 b。 从这里，你可以除以负 11/6 得出 c 等于 3。 所以现在让我们回到起点，现在我们知道 c 等于 3。 然后我们可以去楼上传播。 因此，在第二个方程中替换 c 等于 3，得到 b 等于 2。 现在你知道 b 和 c 在 第一个方程中替换 b 等于 2，c 等于 3，得到 a 等于 4。 因此，最终的解是 a 等于 4，b 等于 2，c 等于 3。
