在本视频中，您将学习如何将 矩阵和向量相乘。 这也是你已经学过的东西，因为它 正是线性方程组的样子。 以下是它的工作原理。回想一下 之前的视频， 两个向量之间的点积是 相应条目的乘积之和。 具有未知变量（ 例如 2a 加 4b 加 c 等 于 28）的方程可以写成 条目 2、 4、1 的行向量和带有未知条目 a、 b 和 c 的列向量的乘积。因此，向量可以开启变量。 这表示一个方程，其中 列向量中的未知数表示 每种水果的价格，行向量中的数字 表示你购买的每种水果的数量。 $28 代表两个向量的点积。 现在，假设你有一个由 三个方程组成的系统，里面有三个未知数。 这些方程中的每一个都可以 表示为点积， 例如，第一个 方程，a 加 b 加 c 等于 10， 可以表示为向量 1、1、 1 的点积， 即 a、b 和 c 的系数，以及带有变量 a、b 和 c 的向量，等于 10 的点积。 方程 a 加 2b 加 c 等于 15， 可以看作是 1、2、1 和 a、b、 c 的点积，那就是 15。 最后，方程 a 加 b 等于 2c 可以看作是向量 1、 1、2 和向量 a、 b、c 的点积，等于 12。 现在用 三种不同的点积来表示这个似乎很笨拙。 有更好的办法吗？ 答案是肯定的。 我们要做的第一件事就是把它们放成这样。 左边是 三个方程和 三个未知变量的系统，右边是 三个点积。 现在，请注意列向量是相同的。 因此，我们可以简单地 将三个向量合并并得到一个矩阵。 现在我们得到矩阵和向量的乘积。 矩阵和向量的乘积只 不过是堆叠在一起的 3 个点积。 如果你的系统中有更多的方程， 那么你的矩阵就更大了。 但是从现在开始， 我们将把方程组表示 为矩阵乘以向量的乘积。
