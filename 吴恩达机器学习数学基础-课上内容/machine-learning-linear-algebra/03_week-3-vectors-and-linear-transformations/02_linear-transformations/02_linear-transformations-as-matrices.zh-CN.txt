在前面的视频中， 您将学习如何将矩阵转换为线性变换。 但是反其道而行同样容易。 在这里，我将向你展示如何从线性变换开始， 然后找到相应的矩阵。 因此，假设我们有一些隐身矩阵， 我们知道它会将这个基本的正方形或基数发送到这里的这个矩阵中。 目标是找到矩阵的条目。 因此，让我们来看看它把每个积分发送到哪里。 点 0, 0 被发送到 0, 0，这总是会发生。 点 1 0 被发送到 3，-1，即底角。 点 0 1 被发送 2、-3，点 1 被发送到 5 2。 所以事实证明我们不需要那么多信息。 我们这里只需要这两点。 我将用箭头来表示这个点，因为向量往往用 从原点一直延伸到点的箭头来表示。 因此，坐标为 1 0 的向量被发送到坐标 3、-1 处的向量。 坐标为 0 1 的那个被发送到坐标为 2 3 的那个。 现在我邀请你拿一张纸参加马拉松比赛， 说服自己矩阵乘法就是这样运作的。 但是，将向量 1 和 0 1 发送到 3、 -1 和 2 3 的矩阵正是具有第 3、-1 和 2、3 列的矩阵。 因此，这就是你如何将线性变换转换为其对应的 矩阵。 你只看两个基本向量 1 0 和 0 1 的去向， 它们是你的矩阵列。
