在之前的视频中，你已经看到矩阵是 代表线性方程组的数字竞赛。 但是还有另一种非常强大且非常有用的矩阵表示形式 ，它是线性变换。 线性变换是一种以 非常结构化的方式将平面中的每个点发送到平面中另一个点的方法。 在这里，我向你展示如何做。 在这段视频中，你将看到矩阵如何成为 二维线性变换的插图，但我邀请你在更多维度上想象它， 例如三维空间。 假设你有这个二乘二矩阵，条目为 311 和 2。 因此，这是它对应的线性变换。 首先考虑两个标有 a 和 b 访问权限 的平面。变换会通过 以下方式将左侧平面上的每个点发送到右侧平面上的某个点。 任何点都有两个坐标。 这两个坐标构成一个列向量。 为了得到右边的向量，我们将第一个向量 乘以矩阵，得到的就是右边的点。 因此，通过一些示例，这会更容易。 首先，让我们来看看原点中的点。 0 0 变成向量 0 0， 我们将其乘以矩阵，我们得到向量 0 0。 所以 00 变为 00。 线性变换实际上总是会发生这种情况。 原点被发送到原点。 现在让我们来看看点 1 0。 因此，矩阵乘以向量 1 0 给出了向量 3 1。 所以 1 0 变为 3 1。 现在让我们来看看 0 1。 矩阵乘以 0 1 等于 1 2。 所以 0 1 变成 1 2。 最后让我们来看看 1 1。 矩阵乘以向量 1 1 是向量 4 3，因此 1 1 变为 4 3。 实际上，这定义了整个转型。 因此，让我们来看看这四个要点的这个小方形。 它指向这个平行四边形。 左边的正方形被称为基础。 他们右边的平行四边形也是如此。 它们是线性代数中非常非常重要的概念。 稍后你就会知道为什么它们被称为 basis，而 基础的一个非常特殊的特性是它们覆盖了整架飞机。 所以实际上，既然这个正方形实际上是 Tessalit 整架飞机，平 行四边形 Tessalit 也是整架飞机。 然后将线性变换简单地定义为坐标的变化。 因此，举个例子，如果我们想找到点减去 2、 3 的位置，那么左边的点减去 2、3 是通过从 原点开始向左走两个方块，向上走三个方块来获得的。 因此，为了找到该点在线性变换中的位置，我们只需开始 原点，然后在 这些新坐标中向左行走两个方块，向上行走三个方块。 然后我们得到的分数减去 3、4。 因此，矩阵乘以向量减去 2，3 就是向量减去 3、4。 用买苹果和香蕉的语言来说。 我喜欢通过以下方式看到转换。 假设你在第一天去商店买了三个苹果和 一个香蕉。 第二天你买了一个苹果和两个香蕉。 因此，如果苹果的价格是 a，香蕉的价格是 b。 那么第一天和第二天的结果价格就是向量的条目， 对应于这些矩阵的乘积，条目为 3 1 1 2。还有@@ 向量 a b。 线性变换将你从左边 的准入点是苹果的价格和香蕉的价格带到右边的一个点， 那里的访问是你第一天支付的 价格，第二天支付的价格。 因此，苹果是1美元，香蕉也是1美元。 然后，第一天你支付了4美元， 第二天你支付了三美元，对矩阵进行了线性变换，将坐标为 1 1的点发送到坐标为4 3的点。 然后，坐标的变化是苹果和 香蕉的价格与第一天和第二天的价格。
