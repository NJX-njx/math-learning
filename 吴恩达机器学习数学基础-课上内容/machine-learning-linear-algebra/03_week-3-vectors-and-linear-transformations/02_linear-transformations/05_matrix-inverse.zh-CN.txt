矩阵可以有一个与之关联的非常特殊的矩阵，称为逆矩阵。 当我想到矩阵的逆矩阵时，我会想到一个数字的倒数， 即乘以后得到一个的数字。 例如，数字二的倒数是数字一半，负五 的倒数是负五分之一。 逆矩阵正是矩阵 的乘积为单位矩阵的矩阵。 在线性变换 中，逆矩阵是撤消原始矩阵工作的矩阵， 即将平面返回到起始位置的矩阵。 以下是逆矩阵的工作原理。 想象一下，你有一个 与带有条目 3、1、 1 和 2 的普通矩阵相对应的线性变换，然后将正方形变成这个平行四边形。 现在有一些变换可以将这个平行四边形变回原来的正方形 。 而且不用担心矩阵的条目，我们稍后会找到它们。 只要知道有一个，那就意味着两个 变换的组成是对应于单位矩阵的变换，对 平面不起任何作用的变换。 它会自行离开。 因此，如果逆矩阵是带有条目 a、b、c 和 d 的矩阵，那就意味着 当我们将原始矩阵乘以逆矩阵时，我们得到恒等式。 现在，就像数字一样，我们要调用逆矩阵，将 原始矩阵提升到-1的次方。 就像你说 2 对负一等于二分之一， 那么这个矩阵到 -1 就是反矩阵。 这恰好有这些条目，你可以验证如果你将 原始矩阵乘以这个矩阵，你就会得到单位矩阵。 现在我们如何找到这个逆矩阵中的条目呢？ 答案是求解线性方程组。 因此，请注意，如果我们将 左边这个矩阵中的条目等于右边的矩阵中的条目， 那么我们实际上有以下四个点积。 这些点积中的每一个都给了我们一些线性方程。 现在你有一个由四个线性方程组成的系统，其中包含四个未知数， a、b、c 和 d。 我们可以使用我们常用的消除方法来求解它，得到 a = 2/5、 b = -1/5、c = -1/5 和 D = 3/5。 所以现在这里有一个测验。 求出以下矩阵的逆矩阵。 如果您发现任务不可能，请随时单击 “我找不到”。 因此，通过求解相应的线性方程组，我们可以得到以下结果。 我们得到的答案是 a = 1/4、 b = -1/4、c = -1/8 和 d = 5/8。 现在是另一个测验，找出以下矩阵的逆矩阵。 而且，如果你发现任务是不可能的， 可以随时点击我快要死胡同了。 事实证明，反向并不存在。 我们需要求解以下线性方程组 a + c = 1、 2b + 2d = 0、2a + 2c = 0 和 b + d = 1。 现在这显然是一个矛盾，因为第一个方程说 a + c = 1， 第三个方程说 2a + 2c = 0，但是如果 a + c = 1，那么 2a + 2c 必须是 2。 它不能是 0。 因此，这个矩阵没有逆矩阵。
