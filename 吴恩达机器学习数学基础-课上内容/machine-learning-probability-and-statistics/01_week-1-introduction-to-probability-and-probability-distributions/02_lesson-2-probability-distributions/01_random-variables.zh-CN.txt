欢迎来到第二课。 现在我们来谈谈概率中最重要的概念之一，那 就是随机变量的概念。 而且这不是什么新鲜事物，你以前见过变量。 例如，在代数中，你可能已经学会了变量 x 等于 3， 它总是具有相同的值。 随机变量不同，它们可以取许多值。 例如，温度是一个可以取许多值的随机变量。 另一个随机变量是我掷硬币 10 次时获得的头数。 这是另一个随机变量。 因此，在这段视频中，我将向大家展示所有关于随机变量的知识。 因此，让我们回到掷硬币的实验。 当你掷硬币时，假设你可以获得正面或反面。 假设你可以 像以前一样以二分之一的概率获得两者。 现在，让我们稍微改变一下。 让我们用一个变量 X 来称之为正面数。 那么现在当你掷硬币时会发生什么，我们能得到多少个头呢？ 好吧，如果我们获得正面，那么我们获得 1 个正面，如果我们获得反面， 那么我们获得 0 个正面。 如果我们看 X，那么如果我们获得正面一次，X 可以是 1； 如果我们获得 0 次正面，X 可以是 0。 而且 X = 1 的概率与获得正面的概率相同， 即 0.5。 而且 X = 0 的概率与获得反面的概率相同， 即 0.5。 X 被称为随机变量， 是概率和统计学中最重要的概念之一。 你可以把 X 看作是一个不一定具有相同值的变量。 有一半的时间它需要的值为 1， 一半的时间则大致为 0。 现在让我们来看看另一个随机变量。 让我们来看看我们在 10 次掷硬币中获得的正面数量。 因此，假设我们把它翻了十次，它们都是正面，所以 X = 10。 或者我们翻转它们，其中一个是尾巴，所以我们得到 X = 9。 我们可以通过许多不同的方式获得 X = 9。 假设我们坚持正面概率等于 0.5。 那么这些事件的概率是多少？ 好吧，由于它们都是独立的，所以它们都有一半的概率， 那么这是十的二分之一。 那第二个呢？ 好吧，现在也是 10 号的一半。 但是我们可以把它写成所有正面的 9 的一半， 剩下的尾巴写成一半，下一个尾巴写成同样的东西。 但是，一个更难的问题是真正问一下我们 总共获得 0 个正面的概率有多大？ P (X = 0) 或 P (X = 1) 或者在 P (X = 10) 之前的所有值，因为在 左边我们有一个特定的场景，一种获得 正面和反面的特殊方式，但在右边我们把它们全部分组。 如你所见，第一个是一样的。 P (X = 0) 等于获得所有正面，即 0.5 比 10。 但是其他的要难一点。 因此，当我想计算东西时，我要做的第一件事就是做一个实验。 因此，让我们来做个实验。 因此，假设我们多次运行这个实验，我们 又得到了 X = 5、X = 6、X = 5，然后我们运行了很多次。 可能的结果是什么？ 好吧，可能的结果是 0 一直到 10，因为你可以获得 0 个正面 ，1，正面一直到 10。 而且，如果我们坚持正面概率等于0.5， 假设我们要重复这个 500 次，让我们看看会得到什么。 我们得到的直方图就是这个。 请注意，这只是一个直方图，不是实际的概率。 但是它可以很好地估计概率，因为我们运行了 很多次。 请注意，X = 0 或 X = 10 的概率是最小的， 因为我们不太可能获得 10 个正面或 10 个反面。 最高的是这里，P (X = 5)，因为我们很可能会得到 大约一半，我们可以看到所有其他的。 例如，这里是 P (X = 8)，那就是你获得 8 个正面和 2 个反面的时候。 因此，这是一个随机变量。 为什么我们关心随机变量？ 因为它们允许我们同时对整个实验进行建模。 而且大多数概率问题都可以用随机变量来表示。 因此，举个例子，如果你掷一堆硬币，正如我们所见， X 是正面数量，那就是一个随机变量。 你可以掷一堆骰子然后将 X 定义为获得 1 的次数。 或者你可以看到一群病人然后说 X 是病人的数量。 或者你可以直接定义随机变量。 假设我定义的概率为 1，概率为一半，但它也是 -7，概率为 0.2，也是 3.14159，概率为 0.3。 我可以想出任何我想要的随机变量来取任何值。 只要概率加起来为 1，我就很好。 现在，这里有其他随机变量。 例如，如果您在等公交车，则下一辆公交车 到达之前的等待时间是一个随机变量，可能需要许多不同的数字。 体操运动员的跳跃高度也是一个随机变量，具有一定范围的值。 货件中缺陷产品的数量也是一个随机变量。 假设 11月的降雨量毫米也是一个随机变量。 你可能会注意到它们的行为方式不同， 这是因为随机变量有两种大类型。 有离散和连续的随机变量。 到目前为止，我们已经看到的是离散的，如果我 掷骰子，则为1或6秒，如果我掷硬币，则为正面数量， 特定群体中具有特定身高或特定年龄的孩子的数量。 例如，连续的可能是距离 下一辆公共汽车到达的时间、体操运动员的跳跃高度、特定月 份的降雨毫米数。 它们为什么不同？ 我鼓励你看一看，告诉我为什么你认为它们与众不同。 发生的一件事是，左边的可以取有限数量 的值。 我可以掷硬币十次然后我可以一直获得 0 到 10 个正面。 另一方面，右边的值可以是无限数量的。 因为时间可以是 1 分钟， 但也可以是 1.01 分钟或 1.001、2、3 分钟。 他们可以取的值数量是无限的。 但实际上事实并非如此 ，离散值也可以取无限数量的值。 我给你举个例子。 如果我开始掷硬币直到获得正面，我可以掷一次、 两次、三次、四次。 我能做到 500 或 100 万但可能无法获得人头。 概率很小，但是我的值范围是无限的。 因此，真正发生的事情是，离散值只能取可 数的值，而连续的值只能取整个间隔内的值。 因此，离散随机变量可以取的值可以放在列表 1、 2、3、4、5 等中。 而连续随机变量取的值不能放在 列表中，因为它是整个间隔。 现在你可能在想，这些随机变量 和你在代数和微积分中看到的变量有什么区别？ 不同之处在于，你在代数和 微积分中看到的是确定性的，而我们在这里看到的是随机的。 什么是确定性？ 例如，x = 2 或者例如， 函数的输入 f (x) = x 平方，它们总是取相同的值。 一旦定义完毕，它就会永远如此。 但是，随机变量不是那样的。 例如，它可以采用许多值。 它们与不确定的结果有关。 因此，确定性变量与固定结果相关联， 随机变量与不确定结果相关联。 这就是主要的区别。
