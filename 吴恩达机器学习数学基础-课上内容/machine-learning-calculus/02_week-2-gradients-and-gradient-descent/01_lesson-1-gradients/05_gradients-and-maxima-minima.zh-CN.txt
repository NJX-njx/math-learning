因此，在这段视频中，我将向你展示一些非常有趣的东西。 梯度对于最小化两个或多个变量的函数非常 有用，就像导数可用于最小化一个变量的函数一样。 因此，让我们回过头来看一个变量的函数。 这个比喻，x 的 f 等于 x 的平方。 现在让我们来看看我们在两个变量中研究的函数， 即 f (x, y) 等于 x 平方加 y 平方。 现在，让我们回到一个变量的函数。 当你想最小化这个函数时，最小点就是这里的这个点。 而找到它的方法是找到斜率等于0的点。 因此，我们取导数，求解导数等于 0， 解就是这个最小点在 0 处。 那么，如果变量越多，会发生什么呢？ 好吧，非常相似的东西。 最小点就是这里的这个点。 请注意，切线平面实际上与地板平行， 并且在每个局部最小值和最大值中都会发生这种情况， 即平面与地板平行。 现在，既然我们不是在处理平面，而是已经完成了偏导数，所以 让我们实际画出两个偏导数。 请注意，它们是这两条线，两条线 的斜率都为 0，因为它们与地板平行。 因此，换句话说，当 偏导数给出的切线的斜率均为 0 时，就会出现右边函数 f 的最小值。 而且，如果你有一个由许多其他变量组成的函数，比如一个由 12 个 变量组成的函数，那么你只需要让 12 个斜率变为 0。 对应于所有偏导数的那些。 因此，现在让我们来做一点数学运算。 在左边，你想求解斜率等于 0，因此导数等于 0。 这意味着你要求解 x 的 f 素数等于 0。 现在，x 平方的 x 素数为 2x， 因为那是 x 平方的导数。 当你求解 2x = 0 时，你基本上得到的是解 x = 0。 而且 x = 0 正是这里的重点。 现在，当我们在更多变量中执行此操作时会发生什么？ 好吧，你希望两个偏导数为 0。 这意味着 dx 和 dy 的数据均为 0。 现在，df 比 dx 有什么不同，那是 2 倍。 而且 df over dy 是 2y。 所以你想要 2x = 0 和 2y = 0。 这与点相同，x，y 等于 0，0，也就是这里的这个点。 现在，这一次，求解方程组，2x = 0， 2y = 0，很容易得到点 x，y = 0。 有时你可能需要稍加努力， 但基本上，要找到最小值和最大值，你所要做的就是将所有 偏导数设置为0并求解该方程组。
