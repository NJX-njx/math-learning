所以现在假设你有两条电源线，一条是蓝色的，一条是橙色的。 作为参考，这里是起源。 因此，蓝色的电源线距离原点只有一米远。 橙色电源线距离原点 b 米， 你的房子距离原点 x 米。 关键是要找出房屋的位置， 以最大限度地降低连接两条电力线的成本。 因此，从房子到蓝色电源线的距离是 x 减去 a。 从房子到橙色电源线的距离 将是 b 减去 x。 所以，连接到蓝色电源线的成本 将是距离的平方，即 x 减去一个正方形。 连接橙色电源线的成本将是那个距离的 正方形，也就是b减去x的平方。 现在，请注意 b 减去 x 的平方等于 x 减去 b 的平方， 因为负数的平方是正数 2 的平方。 因此，我们将把这个距离视为 x 减去 a、 x 减去 b、x 减去 c 等。 即使它们是负数 ，也没关系，因为正方形是正确的。 那么现在，连接两条电源线的总成本是多少？ 好吧，就是这两个成本的总和，x 减去 a 的平方，加上 x 减去 b 的平方。 如果 x 在这里，那么问题的重点是找到距离 x， 这将最小化这个成本函数。 因此，剧透警告，解决方案是将其放在两 条电力线的最中间以降低成本，你很快就会看到这一点。 但首先，让我们尝试将问题可视化，以更好地理解为什么房子 应该放在中间。 因此，让我们实际将成本可视化为正方形的面积， 因为正方形的面积是 x 减去平方。 这就是连接蓝色电源线的成本。 连接橙色电源线的成本由 此处的正方形面积表示，即 x 减去 b 的正方形。 而这两个区域的总和，或 总面积，就是连接两条电力线的总成本。 这就是我们应该最小化的函数。 所以现在让我们来看看一些案例，看看房子是否在某个地方， 面积的总和不是那么大。 如果我们把它放在左边太多，那么橙色方块就太大了。 而且，如果我们把它放在右边太多，那么蓝色方块就太大了。 因此，中间的某个地方应该是理想的位置是有道理的。 但是，让我们用数学来找到理想的位置。 因此，成本函数为 x 减去 a 的平方，再加上 x 减去 b 的平方。 这是一个二次方程，二次方程是这样的， 它们要么指向上方，要么指向下方。 这个特别指向上方， 因为 x 平方系数为正。 换句话说，它是两个 x 平方，再加上其他的东西，再加上其他的东西。 现在，在这个抛物线中，我们必须找到最小点。 这是导数为零的 点，切线的斜率为零的点。 因此，这条切线是水平的，我们可以通过一些计算来找到它。 因此，让我们来看成本函数的导数。 那是 x 的 d x 减去 a 的平方加 x 减去 b 的平方，然后将其 设置为零。 因为当它等于零时，那就是最小点。 因此，通过使用链式法则， 你可以看出 x 减去 a 平方的导数是 x 减去 a 的 2 倍。 而 x 减去 b 平方的导数是 2 倍 x 减去 b，因此 该和等于零。 你可以除以二得到 x 减去 a，加 x 减去 b 等于零。 现在重组得到 2x 减去 a 减去 b 等于零， 这等于 x 等于 a 加 b。 所以，x 等于 a 加 b 大于 2。 因此，最佳解决方案是将房屋放在 两条电力线的最中间，以最大限度地降低成本。 因此，现在，让我们用更多的电力线来做。
