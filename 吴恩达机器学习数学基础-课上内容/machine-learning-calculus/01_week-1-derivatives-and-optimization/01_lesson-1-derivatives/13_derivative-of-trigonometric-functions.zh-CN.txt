让我们来看看另一个函数家族 ，即三角函数。 让我们从正弦函数开始。 让我们考虑几个特别的观点。 减去 Pi 大于 2， Pi 大于 2，0，再减去 Pi。 现在让我们来看看斜坡。 Pi 超过 2 处的 斜率为 0，减去 Pi 大于 2 处的斜率为 0，0 处的 斜率实际上为 1 ，负 Pi 处的斜率将为负 1， 即左边的那个。 现在让我们将其与函数余弦进行比较。 现在我们不打算看斜坡， 而是要看数值。 当 Pi 大于 2 时， cos 的值为 0， 减去 Pi 大于 2 时为 0。 在 0 处为 1，在负 Pi 处为负 1。 你认为这是巧合吗？实际上，事实并非如此。 如果 x 的 f 是 x 的正弦值， 那么 x 的导数 f 素数是 x 的余弦值。 为什么会发生这种情况？ 我们过一会儿再来看看， 但这是一处非常非常不错的房产。 现在， 除了减号之外，同样的事情以相反的方式发生。 如果这是余弦， 让我们来看看一些斜率。 在 0 减去 Pi 时， Pi 大于 2，减去 Pi 大于 2。 在 0 时，斜率为 0， 在负 Pi 时，斜率再次为 0。 Pi 大于 2 时为负 1 ，负 Pi 大于 2 时为 1 现在让我们来看看 x 的正弦值， 但现在是值。 在 0 时你得到 0， 在负 Pi 时你得到 0。 当 Pi 大于 2 时你得到 1，在 负 Pi 超过 2 时你得到负 1。 差不多一样，我们只需要 用一个牌子把它翻过来即可。 如果 x 的 f 是 cos x， 则导数是 x 的负正弦值。 再说一遍，我们稍后就会明白为什么， 但这是要记住的。 正弦的导数是余弦 ，余弦的导数是负正弦。 它起 作用有一个很好的几何理由，看看吧。 让我们画一个半径为 1 的圆然后画 任何径向线，x 是 水平轴和直线之间的角度。 x 轴上的投影是 余弦效应， y 轴上的投影是 x 的正弦值。 现在让我们稍微移动角度 x 的 量 Delta x。 现在水平轴上的投影将是 x 加上 Delta x 的余弦值， 垂直轴上的投影将是 x 加上 Delta x 的正弦值。 现在让我们专注于这个橙色三角形， 让我们把它放大一点。 这个三角形有一个基数等于 x 的负 Delta 余弦值， 另一边是 x 的 Delta 正弦值。 现在我们可以将顶角称为 Phi 和 斜边 h。如果你回想起三角学， 有一条规则指出，给定一个 直角三角形，其中一个角度的余弦等于相邻边除以斜边。 这里的相邻边是 Delta 正弦值 x， 而斜边是 h。现在假设 Delta 正弦值 x 等于 h cos Phi。 同样，有一条规则规定 正弦 等于正弦除以斜边。 在这个三角形中，对 面是负 Delta cos x， 而斜边又是 h。 因此，负 Delta cos h 是 Theta 的 h 正弦值。 现在，随着 delta 越来越小，会发生什么？ 好吧，这个三角形的斜边几乎 与圆拱重合，接近 x 的 Delta。 角度 Phi 接近 x，有了这个新值， 我们得到 Delta 正弦 x 等于 Delta x cos x， 负 cos x 等于 Delta x 正弦 x。同 样 Delta 正弦 x 除以 Delta x 是正弦的导数，而 Delta cos x 除以 Delta x 是 x 的导数。 这实际上证实了你之前建立的直觉。 x 的正弦素数为 cos x ，x 的余弦素数为负正弦 x。
